Roadmap
=======

This is a living document - make sure you're on the latest Roadmap.

### Alpha Deliverable

First "release" - possibly end of the year. Notable features:
 * Register, login, and see active games
 * Start a board game with up to 4 players
 * Display a board game with 4 unique player identifiers
 * Proper updating of boardgame state
 * Basic functionality - moving, choosing directions, etc.
